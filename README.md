# Database-management system (DBMS)
Database management system(DBMS) using Bash-Script
A simple Database Management System using bash shell scripting that handles all CRUD operations.
# Features
- Create DB
- Connect DB
- Drop DB
- List All DB 
- Create Table
- Insert into tables
- Delete from Table
- Drop Table
- Select from Table
- Update Table
- List All Tables
